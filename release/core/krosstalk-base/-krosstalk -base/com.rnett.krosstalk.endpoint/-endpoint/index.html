Class representing an endpoint template. Valid template structures:

  • static text

  • {var} -> replace w/ value

  • {{var}} -> replace w/ /var/{var} or &var={var} if in query params

  • [?var:...] -> body if var != null, else empty. The body must contain whole segments

  • {{?var}} -> [?var:{{var}}]
