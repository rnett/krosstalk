Makes a method a krosstalk method. Should only be on the expect declaration if using a common Krosstalk.

By default, the endpoint will be "/krosstalk/$methodName" although the prefix can be overridden in Krosstalk.

Ordinarily, a hash of the parameters will be included in the method name to support overloads. This can be disabled by setting noParamHash to true, but an compile time error will happen if this is set on a method with overloads that are also krosstalk methods on the same class.
