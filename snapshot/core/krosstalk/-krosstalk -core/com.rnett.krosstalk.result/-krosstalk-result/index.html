KrosstalkResult

interface KrosstalkResult<out T>
Content copied to clipboard

The result of a krosstalk call. Can be either an exception in the server method, a Http error, or success.

Can be handled using a when block (including using Failure, SuccessOrServerException, and SuccessOrHttpError, since everything is sealed), or using the provided recover*, handle*, and throw* functions. Will confine the type to SuccessOrServerException or SuccessOrHttpError when possible (i.e. after a throws* function).

To use as a return type of a Krosstalk method, use with @ExplicitResult.
